# Git Remote
- git remote add origin https://github.com/joniheri/adonisjs-test-api.git

# Requirement
- Node JS 14.x.x
- Pastikan Adonis/CLI sudah terinstall di komputer
- Jika Adonis/CLI belum terinstal, silahkan ketikkan ini di terminal : "npm i -g @adonisjs/cli"

# cara membuat project baru
- Pastikan anda sudah masuk ke directory yang akan anda jadikan tempat penyimpanan project
- Karena kita hanya akan membuat API, Kemudian silahkan buka di directory tempat penyimpanan project anda nanti, kemudian ketikkan ini : "adonis new app project-name --api-only"

# Run This Project
- pastikan sudah di directory project dg cara "cd /project name"
- kemudian untuk runing project, ketikkan ini "adonis serve --dev"

# Untuk membuat controller pada folder ./app/Constrollers dan di folder ./app/Model
- ketikkan pada terminal "adonis make:model ModelName -c"
